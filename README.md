Flashcard Quiz App

Overview

This project is a Flashcard Quiz App developed as part of the CodeAlpha App Development Internship. 
It helps users study using digital flashcards that can be created, edited, and deleted.

Features

* Add new flashcards
* Edit existing flashcards
* Delete flashcards
* Show and hide answers
* Navigate between flashcards using Next and Previous buttons
* Simple and user-friendly interface

Technologies Used

* HTML
* CSS
* JavaScript

Project Structure

* index.html – Application structure
* style.css – Styling and layout
* script.js – Application functionality

How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in a web browser.
